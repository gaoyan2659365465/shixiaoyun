# 🚀 首页快速启动指南

## ⚡ 5分钟快速启动

### 步骤1: 准备图片资源（2分钟）

创建以下目录并放置图片：

```bash
static/images/home/
├── banner/
│   └── banner1.jpg          # 任意图片，建议750×560px
├── icons/
│   ├── checkin.png          # 任意图标，建议128×128px
│   ├── cooperation.png
│   └── points.png
└── activities/
    └── activity1.jpg        # 任意图片，建议750×800px
```

**临时方案**：可以先使用占位图片，后续替换。

### 步骤2: 运行项目（1分钟）

```bash
# 在项目根目录执行
npm run dev:mp-weixin
```

### 步骤3: 打开微信开发者工具（1分钟）

1. 打开微信开发者工具
2. 导入项目
3. 选择 `dist/dev/mp-weixin` 目录
4. 查看首页效果

### 步骤4: 测试功能（1分钟）

- ✅ 查看轮播图
- ✅ 点击快捷入口
- ✅ 查看活动列表
- ✅ 下拉刷新
- ✅ 上拉加载

## 🔧 常见启动问题

### Q1: 找不到组件？

**原因**: 组件路径错误

**解决**: 检查组件导入路径是否正确

```javascript
// 正确的导入方式
import SearchBar from './components/SearchBar.vue'
```

### Q2: 样式不生效？

**原因**: 样式变量未导入

**解决**: 确认 `uni.scss` 第一行是：

```scss
@import '@/styles/variables.scss';
```

### Q3: 页面空白？

**原因**: 数据加载失败

**解决**: 
1. 检查 `data/home.js` 是否存在
2. 查看控制台错误信息
3. 确认图片路径正确

### Q4: 点击无反应？

**原因**: 跳转页面不存在

**解决**: 暂时注释掉跳转代码，或创建对应页面

```javascript
// 临时方案：显示提示而不跳转
handleSearch() {
  uni.showToast({
    title: '搜索功能开发中',
    icon: 'none'
  })
  // uni.navigateTo({ url: '/pages/search/search' })
}
```

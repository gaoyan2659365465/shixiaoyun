# 视消云项目需求文档

## 1. 项目概述
微信小程序 + 后端系统

## 2. 技术栈

### 2.1 前端技术栈
- **开发工具**: HBuilder X
- **框架**: uniapp
- **语言**: Vue3

### 2.2 后端技术栈
- **语言**: Java 8
- **核心框架**: Spring Boot 2.7.18
- **微服务框架**: Spring Cloud 2021.0.9
- **数据库**: MySQL 5.6
- **ORM框架**: MyBatis-Plus
- **构建工具**: Maven
- **API文档**: Springdoc OpenAPI (基于Swagger)
- **消息队列**: RabbitMQ
- **缓存**: Redis

### 2.3 安全要求
- 账号密码加密存储
- 密码采用可逆加密（便于数据迁移）
- 交付时提供密码解密方式

## 3. UI设计规范

### 3.1 颜色规范

**品牌色与辅助色**
- 品牌蓝: #4458ff
- 辅助蓝: #1386ff
- 辅助红: #f34545
- 辅助橙: #ff7437
- 辅助黄: #ff9317
- 成功提示: #0fbc91

**中性色**
- 背景色: #F5F6FA
- 分割线/卡片背景: #E5E6EA
- 边框: #D8D9DC

**字体色**
- 标题文字: #17171a
- 正文: #656a73
- 辅助: #a7abb2

### 3.2 字体规范

**系统字体**
- iOS: 中文 PingFang SC，数字 Helvetica
- Android: 中文思源黑体，数字 Roboto

**字阶定义**
- T1: 24pt/字重600/行高32pt - 特大标题
- T2: 18pt/字重500-400/行高26pt - 大标题、顶部导航
- T3: 16pt/字重500-400/行高22pt - 正文
- T4: 14pt/字重400/行高20pt - 辅助说明
- T5: 12pt/字重400/行高16pt - 辅助级文案
- T6: 10pt/字重400/行高14pt - 备注级文案

### 3.3 布局规范

**画板尺寸**: 375×812

**栅格系统**: 基数为4，所有padding、margin、sizing均为4的倍数

**间距规范**
- W4 (16px): 界面内和屏幕左右间距，可点击icon之间间距
- W3 (12px): 左图右文结构，图片文字左右间距
- W2 (8px): 左icon右文字结构，icon文字左右间距
- W1 (4px): 元素之间最小间距

**卡片规范**
- 圆角: 8px
- 边框: 16px
- 卡片内元素上下间距: 4px

**弹窗规范**
- 圆角: 8px
- 左右边框: 24px
- 上边框: 20px
- 下边框到按钮间距: 24px
- 弹窗内元素上下间距: 16px
- 确定按钮高度: 56px

**表单/输入**
- 总高: 56px

### 3.4 组件规范

**按钮**
- 类型: 主要按钮、中按钮、小按钮、次要按钮、按钮组
- 状态: 正常、点击（增加10%黑色蒙版）、加载、不可点击

**导航**
- 底部Tab: 4栏（首页、场所、消息、我的）
  - 间距: 28px
  - 导航按钮宽: 56px
- 顶部导航: 支持两栏、三栏、四栏
- 导航类型: 一级导航栏、二级导航栏、小程序

**数据录入**
- 搜索框、搜索框+取消
- 表单输入类型:
  - 带右箭头输入
  - 必须输入带星号
  - 带叉号的输入
  - 输入错误带感叹号（下方红色错误提示）

**数据展示**
- 数字标签: 红点、1、99、99+
- 小标签: 蓝色、绿色、橙色、红色、灰色
- 中标签: 蓝色、绿色、橙色、红色
- 大标签: 选中、未选中
- 通告栏:
  - 提示通告栏（带喇叭消息推送）
  - 成功通告栏（带绿色对钩）
  - 警示通告栏（带红色感叹号）
- 头像: 4种不同尺寸

**列表**
- 类型: 纯文字单行列表、说明单行列表、图标+说明单行列表、双行纯文字、图标+双行纯文字、多选列表、单选列表
- 列表分区一标题: 间距8px，宽60px
- 列表分区二标题: 与一标题间距24px
- 说明文字: 最多不超过2行

**操作反馈**
- 顶部功能气泡
- 底部菜单气泡
- 底部弹窗
- 常用弹窗
- 两项操作弹窗
- 输入弹窗
